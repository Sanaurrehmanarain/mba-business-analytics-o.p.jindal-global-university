## 📂 Repository Structure

The repository is organized **by course**. Each course folder contains the following subcategories:

* **Discussion Posts** – Weekly contributions to peer and faculty discussions.
* **Learning Journals** – Personal reflections and weekly summaries.
* **Written Assignments** – Essays, case studies, and other graded submissions.
* **Projects** – Group or individual analytical projects and reports.

